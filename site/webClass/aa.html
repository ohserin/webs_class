• text-decoration-line : text-decoration-line 속성은 글자 라인 스타일을 설정합니다.
  • text-decoration-style : text-decoration-style 속성은 text-decoration-line이 있는 요소에 설정된 선(밑줄, 윗줄 및 선 통과)의 스타일을 설정합니다.
  • text-indent : text-indent 속성은 텍스트를 담고있는 블록 컨테이너에 적용되며 문단의 첫줄을 들여쓰기 할 때 사용합니다.
  • text-justify : text-justify 속성은 텍스트의 간격을 정의합니다.
  • text-orientation : text-orientation 속성은 콘텐츠 줄 내에서 문자의 방향을 설정합니다.
  • text-overflow : text-overflow 속성은 박스 안에 내용이 넘칠 때 텍스트를 어떻게 처리할지 지정하는 속성입니다.
  • text-rendering : text-rendering 속성은 텍스트를 렌더링할 때 최적화할 항목을 선택할 수 있는 속성입니다.
  • text-shadow : text-shadow 속성은 텍스트에 그림자를 추가할 때 설정합니다.
  • text-transform : text-transform 속성은 대문자로 또는 소문자로 바꾸는 속성입니다.
  • top : top 속성은 각 옵션 속성들의 부모 태그를 기준으로 수직으로 위치를 설정합니다.
  • touch-action : touch-action 속성은 어떤 요소 내에서 브라우저가 처리할 터치 액션의 목록을 설정합니다.
  • transform : transform 속성은 요소에 회전, 크기 조절, 기울이기, 이동 효과를 설정합니다.
  • transform-origin : transform-origin 속성은 transform 속성과 함께 사용되는 속성으로, 회전 중심(원점·기준점)을 설정합니다.
  • transform-style : transform-style 속성은 자식 요소를 3D 공간에 배치하거나 2D 평면에 평평하게 배치 할지 설정합니다.
  • transition : transition 속성은 특정 조건 하에서 변화하는 과정을 애니메이션으로 보여주고자 할 때 설정합니다.
  • transition-delay : transition-delay 속성은 프로퍼티가 변화한 시점과 트랜지션이 실제로 시작하는 사이에 대기하는 시간을 초 단위(s) 또는 밀리 초 단위(ms)로 설정합니다.

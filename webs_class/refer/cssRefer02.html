Block와 Inline

Block은 해당 라인 전체의 영역을 가지는 요소, Inline은 내부에 포함된 컨텐츠 만큼의 영역을 가지는 요소입니다.

Block 요소의 특징

줄바꿈 현상이 일어남
너비(width)와 높이(height)를 조절할 수 있음
상하 배치 작업이 가능함

Inline 요소의 특징은 Block 요소와 반대이며, 흐르듯이 옆으로 늘어나게 하는 요소입니다.

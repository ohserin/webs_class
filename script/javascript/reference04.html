식별자/키워드

식별자란?

프로그래머가 변수, 함수 또는 클래스를 정의하기 위해 만든 이름을 식별자라고 합니다.
식별자는 이러한 엔티티를 고유하게 식별하는 데 사용됩니다. 코드를 쉽게 읽을 수 있도록 식별자에 의미있는 이름을 지정하는 것이 중요합니다.
또한 다른 프로그래머가 프로그램의 내용을 식별하는 데 도움이됩니다.


식별자 이름 규칙

식별자를 만들 때 특정 규칙이 있습니다. 식별자는 알파벳 문자, 숫자 및 밑줄 만 사용할 수 있습니다. 숫자로 식별자를 시작하지 않는 것이 좋습니다.
int number = 4와 같은 문장이있을 때; 숫자는 식별자입니다. 프로그래머는 '숫자'라는 이름을 사용하여 해당 변수의 값을 인쇄 할 수 있습니다.
대부분의 프로그래밍 언어는 대소 문자 구분을 지원합니다. 따라서 변수 명‘area’는‘AREA’와 다릅니다.

1. 특수문자 (% * $ @ ^ 등), 공백(탭, space 등)은 식별자로 사용할 수 없으나 '_ ', '$' 는 예외로 사용할 수 있다.
2.'_' 와 '$' 는 식별자의 첫 번째 문자로 사용할 수 있으나 일반적으로 잘 사용하지 않는다.
3. 한글도 식별자로 사용 가능하다.
4. 자바 언어의 키워드는 사용할 수 없다.
5. 식별자의 첫번째 문자로 숫자는 사용할 수 없다.
6. true, false, null은 식별자로 사용할 수 없다.
7. 유니코드 문자를 사용합니다.
8. 공백이 없는 한개의 단어로 구성되어야 한다.
9. 길이 제한이 없다.
10. 대,소문자를 구분한다.


키워드란?

키워드는 특정 작업을 위해 프로그래밍 언어로 제공됩니다. 특별한 의미가 있습니다. 키워드는 식별자로 사용할 수 없습니다.
키워드는 예약어. 프로그램에 int number = 2와 같은 명령문이있을 때; 이는 숫자가 값 2를 포함하는 변수임을 의미합니다. int는 키워드입니다.
메모리 위치가 정수 값을 저장할 수 있음을 컴파일러에 알립니다.
부동 영역으로 명령문이있는 경우 float는 키워드이고 area는 식별자입니다. 영역 변수는 부동 소수점 값을 보유 할 수 있습니다.


자바에서 키워드는 모두 소문자로 되어있다.


기본데이터

boolean, byte, char, shor, int, long, float, double, void


변수

transient, volatile


데이터값

true, false, null


흐름제어

if, else, switch, case, default, for, while, do, break, continue, return


접근제어

public, protected,private


클래스

class, interface, static, abstract, final, new, instanceof, this, super


패키지

package, import


예외처리

try, catch, finally, throw, throws


사용되지 않는 키워드

const, goto


식별자와 키워드의 차이점


* 식별자는 변수, 함수, 클래스에 대한 사용자 정의 이름입니다.
* 체재
* 식별자는 알파벳 문자, 숫자 및 밑줄로 구성 될 수 있습니다.
* 대소 문자 구분
* 식별자는 대문자 또는 소문자 일 수 있습니다.
* 키워드는 소문자 여야합니다.
* 키워드에는 알파벳 문자만 포함됩니다.
* 키워드는 프로그래밍 언어에서 제공하는 예약어입니다.


이름 붙이는 관습

자바 언어에서는 식별자의 이름 규칙 외에 클래스, 변수, 메소드 등에 이름을 붙이는 규칙을 정하고 있지 않습니다.
다만 소스 프로그램의 가독성(readability)을 높이기 위해 일관된 규칙으로 이름을 붙이기를 권유합니다.


클래스 이름

클래스의 이름은 첫 번째 문자는 대문자로 시작하고 여러 단어가 복합되면 각 단어의 첫 번째 문자만 대문자로 표시한다.
public class HelloWorld{}
class Vehicle {}
class AutoVendingMAchine{}


변수, 메소드 이름

변수와 메소드 이름은 첫 단어는 소문자로 표기하고 이후 각 단어의 첫 번째 문자만 대문자로 표기한다. 이렇게 함으로 변수와 클래스를 쉽게 구분할 수 있다.
int iAge;
boolean isSingle;
String strName;
public int getAge() {}


상수 이름

상수는 이름 전체를 대문자로 표기하도록 권장한다. 다음은 상수를 선언하는 예이다.
final static double PI = 3.141592;

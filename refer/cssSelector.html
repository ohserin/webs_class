기본 선택자

속성 예시 설명
태그 선택자 p {color: #fff} p 태그를 선택하여 글씨 색을 하얀색으로 변경합니다.
클래스 선택자 .class {color: #fff} class 클래스를 선택하여 글씨 색을 하얀색으로 변경합니다.
아이디 선택자 #id {color: #fff} id 아이디를 선택하여 글씨 색을 하얀색으로 변경합니다.
그룹 선택자 div, p {color: #fff} div 태그와 p 태그를 선택하여 글씨 색을 하얀색으로 변경합니다.
전체 선택자 * {color: #fff} 전체 태그를 선택하여 글씨 색을 하얀색으로 변경합니다.

계층 선택자

속성 예시 설명
하위 선택자
(Descendant Selector)		 div p {color: #fff} div 태그의 모든 자식 p 태그를 선택하여 하얀색으로 변경합니다.
자식 선택자
(Child Selector)		 din > p {color: #fff} class 클래스를 선택하여 글씨 색을 하얀색으로 변경합니다.
형제 선택자
(Sibling Selector)		 din ~ p {color: #fff} id 아이디를 선택하여 글씨 색을 하얀색으로 변경합니다.
인접 형제 선택자
(Adjacent Sibling Selector)		 div + p {color: #fff} div 태그와 p 태그를 선택하여 글씨 색을 하얀색으로 변경합니다.

속성 선택자

종류 예시 설명
[name] a[href] a 태그 속성 중에 href가 있으면 선택합니다.
[name="value"] a[href="#header"] a 태그 href 속성의 값이 '#header'이면 선택합니다.
[name~="value"] a[href~="web"] a 태그 href 속성의 값이 'web'을 포함한 요소를 선택합니다.
[name|="value"] a[href|="web"] a 태그 href 속성의 값이 'web'이거나 'web'으로 시작하는 요소를 선택합니다.
[name^="value"] a[href^="web"] a 태그 href 속성의 값이 'web'으로 시작하는 태그를 선택합니다.
[name*="value"] a[href*="web"] a 태그 href 속성의 값이 'web'이 포함되어 있는 태그를 선택합니다.
[name$="value"] a[href$="com"] a 태그 href 속성의 값이 'com'으로 끝나는 태그를 선택합니다.

가상 요소

속성 예시 설명
::first-line p::first-line {color: #fff} ::first-line은 요소의 첫번째 줄을 선택하여 하얀색으로 변경합니다.
::first-letter p::first-letter {color: #fff} ::first-letter 가상요소는, 요소의 첫번째 문자를 선택하여 하얀색으로 변경합니다.
::before p::before {content: '시작'} 요소의 콘텐츠 시작부분에 생성된 콘텐츠를 추가합니다.
::after p::after {content: '끝'} 요소의 콘텐츠 끝부분에 생성된 콘텐츠를 추가합니다.
::selection p::selection {color:red} 요소의 텍스트에서 사용자에 의하여 선택(드래그)된 영역을 빨간색으로 변경합니다.

가상 클래스

속성 예시 설명
:first-child li:first-child {color: #fff} :first-child는 부모 요소 안에 있는 첫번째 리스트를 선택하여 하얀색으로 변경합니다.
:last-child li:last-child {color: #fff} 요소 유형(타입)에 상관없이 요소의 마지막 리스트를 하얀색으로 변경합니다.
:first-of-type p:first-of-type {color: #fff} 부모의 첫 번째 자식이든 아니든 상관하지 않고 첫번째 단락을 하얀색으로 변경합니다.
:last-of-type p:last-of-type {color: #fff} first-of-type과 정확하게 동일하지만 컨테이너의 마지막 단락을 하얀색으로 변경합니다.
:nth-child() li::nth-child(2) {color: #fff} 두번째 리스트 항목의 글자를 하얀색으로 변경합니다.
:nth-last-child() li:nth-last-child(2) {color: #fff} 리스트의 마지막 요소로부터 두번째 리스트를 하얀색으로 변경합니다.
:nth-of-type() p:nth-of-type(3) {color: #fff} 같은 타입(태그)인 자식에 대해서 세번째 자식의 단락을 하얀색으로 변경합니다.
:nth-last-of-type() p:nth-of-type(3) {color: #fff} 같은 타입(태그)인 자식에 대해서 마지막에서 세번째 자식의 단락을 하얀색으로 변경합니다.
:only-child p:only-child {color: #fff} 어떤 요소의 자식이 하나밖에 없는 단락을 하얀색으로 변경합니다.
:only-of-type p:only-of-type {color: #fff} 같은 유형의 형제가 없는 단락을 하얀색으로 변경합니다.
:not() li:not(:nth-child(2)) {color: #fff} 클래스 선택자가 없는 2번째 요소를 하얀색으로 변경합니다.
:root() :root {color: #fff} 특정 위치에 있는 요소를 선택하여 하얀색으로 변경합니다.
:empty :empty {color: #fff} 콘텐츠나 자식이 없는 요소들을 선택하여 하얀색으로 변경합니다.

링크 가상 선택자

속성 예시 설명
:link a:link {color: #fff} 방문 전 링크를 하얀색으로 변경합니다.
:visited a:visited {color: #fff} 방문 후 링크를 하얀색으로 변경합니다. 하얀색으로 변경합니다.
:hover a:hover {color: #fff} 마우스를 오버했을 때 링크를 하얀색으로 변경합니다.
:active a:active {color: #fff} 클릭했을 때 링크 상태를 하얀색으로 변경합니다.

폼 선택자

속성 예시 설명
:checked input:checked {color: #fff} 사용자가 선택한 라디오 버튼이나 체크박스를 하얀색으로 변경합니다.
:focus input:focus {color: #fff} focus를 받으면 하얀색으로 변경합니다.
:enabled input:enabled {color: #fff} 모든 활성화 된 요소를 하얀색으로 변경합니다.
:disabled input:disabled {color: #fff} :disabled를 사용하여 선택할 수 없는 것들을 하연색으로 변경합니다.

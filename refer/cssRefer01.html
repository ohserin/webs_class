CSS란?

CSS [cascading style sheets] 스타일시트
css란 간단히 말해서 디자인이다. 기존의 HTML은 웹 문서를 다양하게 설계하고 수시로 변경하는데 많은 제약이 따르는데, 이를 보완하기 위해 만들어진 것이 스타일 시트이고 스타일 시트의 표준안이 바로 CSS이다. 간단히 스타일 시트라고도 한다.

CSS 주석 표시

코드를 작성하다보면 잠시 일부 코드를 제거하고 실행해보아야할 경우가 있다. 또한 복잡한 코드에 메모를 달아두어 나중에 보아도 이해하기
쉽도록 해야할 필요성을 느낄 때도 있다. 이럴 때 사용하는 것이 '주석'이다 

CSS의 경우에는 /* 주석처리할 내용 */ 으로 주석처리를 해준다.

CSS 선언방법

내부 스타일

HTML과 달리 <body>가 아닌 <head>사이에 사용합니다. 그리고 <style> 내부 css 선언을 하고 그 안에 사용합니다.

<head>
    <style> //태그안에 선언
   내부 css 내용들
    </style>
</head>

외부 스타일

HTML과 달리 <body>가 아닌 <head>사이에 사용합니다. 스타일 시트를 외부 파일로 저장하고 그 파일을 <head>안으로 불러옵니다. 참고로 외부 css를 불러올때는 <style>을 써주지 않고 <link rel="stylesheet" type="text/css" href="내가 지정한 css파일명.css">로 불러옵니다.

<link> //태그로 외부 파일 연결
<head>
    <link rel="stylesheet" type="text/css" href="내가 지정한 css파일명.css">
</head>

인라인 스타일

css중 유일하게 <head>가 아닌 <body>에 들어간다. 인라인css에서 인라인이라면 꾸며 줄 위치에 바로 css를 입력해서 꾸며준다는 뜻이다. <table width="700" style"">이면 style=""이 인라인 css로 ""안에 스타일을 넣어주면 된다.

<body>
    <p style="font-family:나눔바른고딕;"> 안녕하세요 </p>
</body>
